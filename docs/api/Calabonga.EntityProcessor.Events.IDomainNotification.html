Interface IDomainNotification

Уведомление при обработке сущности в процессоре EntityProcessor<TEntity>

Namespace: Calabonga.EntityProcessor.Events
Assembly: Calabonga.EntityProcessor.dll
Syntax
public interface IDomainNotification : IDomainEvent, INotification
Remarks

Уведомления получат все подписчики. Команда IDomainCommand будет обработана одним перехватчиком MediatR.IRequestHandler`2. Уведомления является наследником INotification и выполняется через команду PUBLISH.
